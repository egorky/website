Zammad

La plataforma de tickets moderna para un servicio al cliente excepcional.

Características Principales

  • Integración multicanal: Email, Teléfono, Chat, WhatsApp, Telegram, etc.
  • Base de conocimientos integrada para autoservicio de clientes y agentes.
  • Gestión de Acuerdos de Nivel de Servicio (SLA) y escalaciones.
  • Flujos de trabajo y automatizaciones para optimizar la gestión de tickets.
  • Campos personalizados para adaptar los tickets a tus necesidades.
  • Vistas de tickets personalizables para cada equipo o agente.
  • Historial completo de cambios en cada ticket para auditorías.
  • Chat en vivo para una comunicación instantánea con los clientes.
  • Detección automática de idioma en los tickets entrantes.
  • Soporte para más de 40 idiomas en la interfaz de usuario.
  • Portal del cliente para que puedan seguir el estado de sus solicitudes.
  • Respuestas predefinidas (módulos de texto) para acelerar las respuestas.
  • Búsqueda de texto completo que incluye archivos adjuntos.
  • Registro de tiempo para medir el esfuerzo en cada ticket.
  • Posibilidad de agrupar, dividir y vincular tickets relacionados.
  • Advertencia de conflicto para evitar que dos agentes editen un ticket a la vez.
  • Diseño adaptable (responsive) para una experiencia perfecta en móviles.
  • Importación de datos sencilla a través de archivos CSV.
  • API completa para integraciones con otros sistemas.
  • Personalización de la marca (logo, colores, CSS).
  • Gestión de ausencias con asignación automática a sustitutos.
  • Informes y análisis para medir el rendimiento del equipo.
