GLPI

La solución integral para la gestión de activos de TI y mesa de servicios.

Características Principales

  • Inventario automático de computadoras, software y periféricos.
  • Gestión completa del ciclo de vida de los activos de TI.
  • Mesa de ayuda para la gestión de incidentes y solicitudes.
  • Gestión de problemas para análisis de causa raíz.
  • Gestión de cambios para controlar las modificaciones en la infraestructura.
  • Definición y seguimiento de Acuerdos de Nivel de Servicio (SLA).
  • Base de conocimientos para centralizar la información y soluciones.
  • Gestión de licencias de software para asegurar el cumplimiento.
  • Gestión financiera, de contratos y de proveedores.
  • Módulo de gestión de proyectos con diagramas de Gantt y Kanban.
  • Creación de formularios personalizados para la creación de tickets.
  • Soporte para múltiples entidades para aislar datos de diferentes departamentos.
  • Gestión de perfiles y permisos para un control de acceso detallado.
  • Motor de reglas para automatizar la asignación y gestión de tickets.
  • Notificaciones automáticas por correo electrónico.
  • Generación de informes personalizables sobre activos y tickets.
  • Reserva de equipos y recursos compartidos.
  • Integración con OCS Inventory para un inventario más potente.
  • Autenticación de usuarios a través de LDAP, AD, y otros sistemas.
  • Extensible a través de una amplia gama de plugins.
  • Gestión de centros de datos (DCIM).
  • Seguimiento de consumibles y cartuchos.
